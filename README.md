# prismagram

instagram clone with express + prisma + react and react-native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret(Login)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
